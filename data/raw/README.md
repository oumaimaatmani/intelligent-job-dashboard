# README for Raw Data

This directory contains the raw data collected for the Intelligent Job Dashboard project. The raw data is sourced from various job listing platforms, including but not limited to:

- Indeed
- LinkedIn
- Glassdoor

## Data Structure

The raw data is typically structured in a CSV format with the following columns:

- **Job Title**: The title of the job position.
- **Company**: The name of the company offering the job.
- **Location**: The geographical location of the job.
- **Posted Date**: The date the job was posted.
- **Job Description**: A brief description of the job responsibilities and requirements.
- **Source**: The platform from which the job listing was scraped.

## Usage

The raw data files are intended for internal use only and should be processed using the provided data cleaning scripts before analysis. Please refer to the `docs/data_pipeline.md` for detailed information on the data processing steps.
